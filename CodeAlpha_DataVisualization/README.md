# Superstore Sales Data Visualization

## CodeAlpha Data Analytics Internship Project

### 📌 Project Overview

This project focuses on visualizing Superstore sales data using Python. The objective of this project is to transform raw business data into meaningful visual insights that help understand sales performance, profit trends, customer segments, and regional analysis.

---

## 🎯 Objective

* Analyze Superstore sales dataset
* Create meaningful visualizations
* Identify sales and profit trends
* Understand regional and category-wise performance
* Improve data storytelling skills

---

## 🛠️ Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook / Google Colab

---

## 📂 Dataset

Dataset used: Superstore Sales Dataset

The dataset contains:

* Sales information
* Profit details
* Customer segments
* Product categories
* Regional sales data
* Order dates and shipping details

---

## 🔍 Analysis & Visualizations Performed

### 1. Sales by Region

Visualized sales performance across different regions.

### 2. Profit by Category

Analyzed profit generated by product categories.

### 3. Monthly Sales Trend

Identified monthly sales patterns and trends.

### 4. Top Products by Sales

Highlighted top-performing product sub-categories.

### 5. Sales Distribution

Visualized overall sales distribution.

### 6. Correlation Heatmap

Analyzed relationships between sales, profit, quantity, and discount.

### 7. Segment-wise Sales Analysis

Compared sales contribution from different customer segments.

---

## 📊 Key Insights

* West region generated the highest sales.
* Technology category produced strong profits.
* Consumer segment contributed major sales.
* Discounts impact profit margins significantly.
* Some product categories outperform others consistently.
